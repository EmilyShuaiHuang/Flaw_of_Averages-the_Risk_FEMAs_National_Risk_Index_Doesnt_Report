This repository contains the empirical contribution in Emily Huang's senior thesis "Flaw of Averages: the Risk FEMA's National Risk Index Doesn't Report". The Quarto document uploaded here has all of the code used to reconstruct the NRI and conduct the Monte Carlo Simulation described in the thesis. To render and review the results, replace file path in the quarto document with the uploaded dataset (NRI) and [SHELDUS](https://drive.google.com/file/d/1fY-dTZEYD1-piCMo6dD_DoYXyv2Qt--X/view?usp=drive_link) data set. Contact Emily Huang at eh6041@bard.edu for any question or recommendations.
